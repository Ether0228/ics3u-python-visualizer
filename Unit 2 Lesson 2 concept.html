Python Lists: Core Technical Concepts
Python 列表：核心技术概念

1. Deleting Elements (删除元素)

We can remove items from a list using several methods (我们可以使用多种方法从列表中移除元素):

  • del: Deletes an item at a specific index (删除指定索引处的元素).
  • .pop(): Removes and returns the last item, or an item at a specified index (移除并返回最后一个元素，或指定索引处的元素).
  • .remove(): Removes the first occurrence of a specific value (移除列表中第一次出现的特定值).
a = ["Mia", "Rafer", "Brad", "Alex"]
del a[-1]           # Deletes the last item / 删除最后一项 ("Alex")
a.pop(1)            # Removes item at index 1 / 移除索引 1 处的项 ("Rafer")
a.remove("Mia")     # Removes the specific value / 移除特定值 ("Mia")

2. Updating & Inserting (更新与插入)

You can modify existing data or append new data to the list (您可以修改现有数据或向列表追加新数据):

  • Index Assignment (索引赋值): Modifies the value at a specific position (修改特定位置的值).
  • .insert(): Inserts an item at a given position (在指定位置插入元素).
  • .append(): Appends an item to the end of the list (在列表末尾追加元素).
a = ["Mia", "Rafer"]
a[1] = "Sam"             # Modifies "Rafer" to "Sam" / 将 "Rafer" 修改为 "Sam"
a.insert(1, "Spencer")   # Inserts "Spencer" at index 1 / 在索引 1 处插入 "Spencer"
a.append("Yanbo")        # Appends "Yanbo" to the end / 在末尾追加 "Yanbo"

3. Sorting & Reversing (排序与反转)

Lists can be organized alphabetically, numerically, or in reverse order (列表可以按字母顺序、数字顺序或逆序排列):

  • .sort(): Sorts the elements of the list in place (对列表元素进行原地排序).
  • .reverse(): Reverses the elements of the list in place (原地反转列表元素).
a = ["Mia", "Alex", "Zack"]
a.sort()       # Becomes / 变为 ["Alex", "Mia", "Zack"]
a.reverse()    # Becomes / 变为 ["Zack", "Mia", "Alex"]

4. Searching & Indexing (搜索与索引)

We utilize loops and the `.index()` method to pinpoint the exact location of an element (我们利用循环和 `.index()` 方法来精确定位元素的具体位置):

a = ["Mia", "Brad", "Alex"]
name = "Brad"

if name in a:
    j = a.index(name)
    print("Found at index / 查找到的索引位置:", j)  # Output: 1
else:
    print("Not found! / 未找到！")

5. Advanced: Math and Randomness (进阶：数学与随机性)

We can import Python modules to generate random integers and compute statistical metrics (我们可以导入 Python 模块来生成随机整数并计算统计指标):

import random
import statistics

scores = [90, 89, 70, 45, 67]
random_score = random.randint(50, 100)  # Generates a random integer / 生成一个随机整数

print("Max / 最大值:", max(scores))
print("Min / 最小值:", min(scores))
print("Median / 中位数:", statistics.median(scores))
