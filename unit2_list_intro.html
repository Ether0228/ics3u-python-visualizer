探索 Python 中最强大、最灵活的数据容器！

1. 什么是列表？(与传统数组的区别)

在计算机科学中，如果你需要存储一系列数据的集合，我们通常称之为数组 (Array) [cite: 476]。但在 Python 中，它被称为列表 (List)。

  • 传统语言的限制： 在 C++ 或 Java 等其他语言中，数组通常要求存放相同的数据类型 [cite: 477]。要么全是整数，要么全是字符串。
  • Python 的魔法： Python 列表极其灵活！你可以将不同的数据类型（如字符串、整数、小数浮点数等）混合存放在同一个列表中 [cite: 480, 481, 526, 527]。
# 这是一个混合数据类型的列表 (Atousa老师的 Zoo 案例)
mix = ["monkey", 7, 12, "gorilla"]

2. 内存索引：如何访问列表中的元素？

为了访问列表中的特定元素，我们需要使用索引 (Index)。请记住计算机科学中最重要的法则之一：计数是从 0 开始的！

如果你想访问整个集合 a 中的各个元素，你需要使用 0, 1, 2, 3... 这样的数字 [cite: 489, 490]。

  • 第一个元素是 a[0] [cite: 488]
  • 第二个元素是 a[1] [cite: 488]
  • 第三个元素是 a[2] [cite: 488, 489]

3. 动态扩张：append() 方法

列表是可变的 (Mutable) [cite: 482]。这意味着你可以在程序运行时随时修改它。

如果你想将一个新元素添加到列表的末尾，在 Python 中你只需要使用 .append() 方法 [cite: 556, 559]。这比其他很多编程语言都要简单得多！

4. 遍历列表：超简单的 For 循环

在其他语言中，遍历一个数组可能需要写很长的代码，但在 Python 中，你只需要一句话！

你可以把 k 想象成一个搬运工（容器），它会依次走到列表的每一个元素下面，把当前看到的值装进自己肚子里 [cite: 575, 577, 578]。

for k in a:
    print(k) # 依次打印出 "good news", "you will have", "till Friday" [cite: 575, 578, 581, 583]
